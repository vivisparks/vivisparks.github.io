ぶろぐをつくった

わーいhexoでぶろぐができたよー

やったこと

仮想環境でhexoをうごかす。

調べて出てくるとおり、ローカルで動かすのは1時間もかからずにできた。

GitHub Pagesに公開する。

アカウントはなぜか持ってたけど、まずページの作り方から調べないといけなかった。
公開しようとすると鍵の登録がどうの～とでて身構える。
授業でも苦手で半分こころが折れてたけど、「仮想環境だしバックアップはとったし大丈夫でしょ」とヤケクソ精神でやってみた。
2時間くらいかかってようやく公開。

これがうわさの草を生やすやつ…

プラグインをいろいろ入れてみる。

instagram, twitterの埋め込み
あんまり利用頻度は高くないとは思うけど身近なものなので。

絵文字
がない…やて…
なぜかデフォルトではdisplayがblock要素になっていたのでinlineに書き換え。

でもそう簡単にはいかず2時間くらい詰まる。
configのせい…？と思ったけど、hexoではcssの反映が遅いことがあるらしく

で無事解決。

cssの行も大量にあってﾋｪってなったけど、viのコマンドを覚えてたのでなんとかなった。
「ここ試験にでた！」ってやつ。

まとめ

結局1日かかってしまったけど、楽しかった。
ターミナルで開発やら、gitやら初めてだったけど動くとうれしい。
とりあえずやってみるって大事。意外となんとかなる。

いまはテンプレートほぼそのままなので、見た目をいじりたい。
